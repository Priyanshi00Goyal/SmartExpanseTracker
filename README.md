# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Priyanshi00Goyal/SmartExpenseTracker.git
```

---

## Open Project

```bash
cd SmartExpenseTracker
```

---

## Compile

```bash
g++ main.cpp User.cpp FinanceManager.cpp Utils.cpp ConsoleUI.cpp -o output/main.exe
```

---

## Run

### Windows

```bash
.\output\main.exe
```

### Linux/macOS

```bash
./output/main
```
